# Buttons

Buttons are styled links; they use link syntax.

{{< button relref="/" >}}Get Home{{< /button >}}
{{< button href="https://github.com/cjdbarlow/titfl" >}}Contribute{{< /button >}}

## How To

```tpl
{{</* button relref="/" [class="..."] */>}}Get Home{{</* /button */>}}
{{</* button href="https://github.com/cjdbarlow/titfl" */>}}Contribute{{</* /button */>}}
```
